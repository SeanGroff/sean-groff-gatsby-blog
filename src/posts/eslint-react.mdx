- `> npm i -D eslint eslint-config-react-app @typescript-eslint/eslint-plugin @typescript-eslint/parser babel-eslint eslint-plugin-flowtype eslint-plugin-import eslint-plugin-jsx-a11y eslint-plugin-react-hooks prettier husky lint-staged`
- `> npx eslint --init`
  - How would you like to use ESLint? · problems
  - What type of modules does your project use? · esm
  - Which framework does your project use? · react
  - Does your project use TypeScript? · Yes
  - Where does your code run? · browser
  - What format do you want your config file to be in? · JavaScript

Then extend the recommended eslint config:

```js
{
  "extends": [
    // ...
    "react-app"
  ]
}
```

optional

```js
  rules: {
    // For Next.js projects
    'react/react-in-jsx-scope': 'off',
  },
```

Then in package.json setup husky and lint-staged

```js
  "husky": {
    "hooks": {
      "pre-commit": "lint-staged"
    }
  },
  "lint-staged": {
    "./src/**/*.{js,jsx,ts,tsx}": [
      "npx prettier --write",
      "eslint './src/**/*.{js,jsx,ts,tsx}' --fix"
    ]
  }
```
