## Grid Container

### Display

```css
.container {
  display: grid;
}
```

Creates a block level grid.

### Columns & Rows

```css
.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  grid-template-rows: 20px 20px;
}
```

### Gaps

```css
.container {
  display: grid;
  grid-gap: 8px;
}
```
