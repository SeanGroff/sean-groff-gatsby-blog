# Sean Groff's Personal Site

## Local Development

Run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Lint Code

```bash
npm run lint
```

Upgrade Dependencies

```bash
npm run upgrade
```

## Troubleshooting

Reinstall

```bash
npm run clean
npm install
```
